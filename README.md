# 📊 Purchase Pattern Analytics – Business Intelligence Project

## 📌 Project Overview

This project analyzes customer purchasing behavior, sales trends, product demand, and global distribution using Business Intelligence techniques. The objective is to derive actionable insights that help improve sales performance, inventory planning, and market expansion strategies.

This report provides a comprehensive analysis of:

* Customer distribution
* Sales trends
* Product performance
* Global coverage
* Revenue metrics
* Strategic recommendations

---

## 🎯 Business Objectives

* Analyze purchasing behavior across customers
* Identify top-performing products
* Track monthly sales trends
* Understand geographic customer distribution
* Calculate key performance metrics
* Provide business recommendations

## 📈 Sales Trend Analysis

### Monthly Sales Performance

| Month    | Sales (₹K) |
| -------- | ---------- |
| Dec 2010 | 950        |
| Jan 2011 | 680        |
| Feb 2011 | 520        |
| Mar 2011 | 490        |
| Apr 2011 | 120        |

### Observations

* Peak sales recorded in **December 2010**
* Continuous decline from **January to April 2011**
* Overall **87% drop** after peak month
* April 2011 recorded lowest performance

### Business Insight

Possible reasons:

* Seasonal demand drop
* Supply chain issues
* Marketing reduction
* Customer churn
